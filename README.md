# Password Vault (Python)

A local desktop password manager built with Python, Tkinter, SQLite, and encryption.

## Features
- Encrypted password storage
- Master password unlock
- Add, view, update, delete entries
- Local SQLite database

## Tech
- Python 3
- Tkinter
- SQLite
- cryptography (Fernet)

## Run
pip install -r requirements.txt
python password_vault_ui.py
